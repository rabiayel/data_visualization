# 📊 Data Visualization with Python

Bu repo, Python kullanarak veri görselleştirme tekniklerini içermektedir.

## 📁 İçerik

- `Day2_DataVisualization.ipynb` — Veri görselleştirme uygulamaları
- `Methane_final.csv` — Metan emisyonu verisi
- `corona_dat.csv` — COVID-19 verisi
- `supermarket.csv` — Süpermarket satış verisi
- `train.csv` — Eğitim veri seti

## 🛠️ Kullanılan Kütüphaneler

- Pandas
- Matplotlib
- Seaborn

## 🚀 Nasıl Çalıştırılır?

Jupyter Notebook ile `Day2_DataVisualization.ipynb` dosyasını açın.
